# Retail Sales Analysis Dashboard (Power BI)

## Project Overview
This project focuses on analyzing retail sales data using Power BI to gain meaningful business insights. The dashboard highlights key sales metrics, trends, and comparisons to support data-driven decision-making.

## Tools Used
- Power BI Desktop  
- Microsoft Excel (data source)

## Dataset Description
The dataset contains retail sales information with the following fields:
- Order Date  
- Product  
- Category  
- Region  
- Sales  
- Quantity  

## Key KPIs
- Total Sales  
- Total Quantity Sold  
- Average Sales  

## Visualizations Included
- KPI Cards for key metrics  
- Sales by Category  
- Sales by Region  
- Monthly Sales Trend  
- Top 5 Products by Sales  

## Power BI Concepts Used
- Data cleaning and transformation  
- DAX measures (SUM, AVERAGE)  
- Filters and visual-level sorting  
- Interactive charts and cards  

## Project Outcome
The dashboard provides a clear view of overall sales performance, identifies top-performing products and regions, and highlights monthly sales trends, helping stakeholders understand business performance effectively.
