# Retail Sales Analysis – Project Explanation

## Project Overview
This project focuses on analyzing retail sales data using **Microsoft Power BI** to gain insights into sales performance, customer demand, and product trends. The dashboard helps understand overall business performance through interactive visualizations and key performance indicators (KPIs).

---

## Objective
The main objectives of this project are:
- To analyze total and average sales performance
- To identify top-performing categories, products, and regions
- To understand monthly sales trends
- To present insights through an interactive Power BI dashboard

---

## Dataset Description
The dataset contains retail transaction data with the following fields:
- **Order_Date** – Date of purchase  
- **Product** – Product name  
- **Category** – Product category  
- **Region** – Sales region  
- **Quantity** – Units sold  
- **Sales** – Sales amount  

The data was imported from an Excel file into Power BI.

---

## Data Cleaning & Transformation
Data preparation was done using **Power Query Editor**:
- Promoted headers
- Changed data types (Date, Text, Numeric)
- Verified column consistency
- Removed errors and ensured clean data for analysis

---

## DAX Measures Created
Custom DAX measures were created to support analysis:
- **Total Sales** – Sum of sales amount  
- **Total Quantity** – Total units sold  
- **Average Sales** – Average sales value  

These measures are used in KPI cards and visualizations.

---

## Dashboard Components
The Power BI dashboard includes:
- **KPI Cards**: Total Sales, Average Sales, Total Quantity  
- **Sales by Category** (Bar Chart)
- **Sales by Region** (Horizontal Bar Chart)
- **Monthly Sales Trend** (Line Chart)
- **Top 5 Products by Sales** (Column Chart)

Visual interactions allow users to analyze data dynamically.

---

## Tools & Technologies Used
- Microsoft Power BI  
- Power Query  
- DAX  
- Microsoft Excel  

---

## Key Insights
- Electronics and Clothing are top revenue-generating categories
- Certain regions consistently outperform others
- Monthly trends highlight peak and low sales periods
- A small number of products contribute significantly to total sales

---

## Conclusion
This project demonstrates practical skills in **data cleaning, DAX calculations, and dashboard design using Power BI**. It showcases the ability to convert raw data into meaningful business insights and is suitable for data analyst and business intelligence portfolios.
